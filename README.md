## Introduction
This is a simple console-based calculator project implemented in Python. It allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features
* Basic arithmetic operations: addition, subtraction, multiplication, division
* Console-based interface
* Supports multiple operations in a single session

## Requirements
* Python 3.x

## Installation
1. Clone the repository using `git clone https://github.com/your-username/Console-Calculator.git`
2. Navigate to the project directory using `cd Console-Calculator`
3. Run the calculator using `python calculator.py`

## Usage
1. Run the calculator using `python calculator.py`
2. Follow the on-screen instructions to perform calculations
3. Type 'exit' to quit the calculator
